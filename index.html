ZOUBA

Zouba allows a user to search the available transportation routes from a certain location and save these routes for later on.

This is an HTML5 based project that uses geocoding and the Reittiopas API. It is firstly targeting Android Phones.

To install and run ZouBa :

1) install google chrome on your desktop machine : https://www.google.com/chrome 2) open url chrome://extensions 3) check the box next to 'Developer mode' 4) click 'Load inpacked extension...' 5) select the client/common directory from the git clone 6) click on the 'new tab' button in chrome (little trapezoid next to the top tab) 7) click on the 'ZouBa' icon (you may have to click on a '>' or '<' to get to the page with the extensions on)
